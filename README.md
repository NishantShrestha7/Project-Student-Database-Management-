# Student Database Management System
The purpose of the C++ Student Database Management System. It will load student information that is provided by a text file and will provide a menu that allows the user to view data, search for students, check averages, and look at performace results. 

# What the Program Does
- WIll load student record from a file.
- Display all students including their details
- Allows to serach for Student by:
    - Student ID
    - ZIP code
- Provide class average and student performance
- Sort Student List
- Create a report card for a chosen Student.

# Files in this Project
**project.cpp** - main program and menu
**project.h** - struct declarations and function
**student.txt** - A data file containing student records
